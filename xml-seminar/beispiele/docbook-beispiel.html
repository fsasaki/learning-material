1. Inline Markup and Images

This is a sample showing that <oXygen/> can be used to edit documents in conformity with the dockbookx.dtd.

This is a Docbook figure, inserted using the imagedata tag:

Figure 1. Lake in Fagaras

In order to preview this text in a Web browser, you have to choose the Docbook HTML transformation scenario. For this press the Configure transformation scenario button or the shortcut CTRL+SHIFT+C or (META+SHIFT+C on Mac OS X), then select the scenario. Press Ok.

To apply the stylesheet you have to press the "Apply transformation scenario" button or to press CTRL+SHIFT+T (META+SHIFT+T on Mac OS X).
